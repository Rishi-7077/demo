⚡ Value Iteration

Directly computes the optimal value function V*(s) using the Bellman optimality equation, then derives the policy.

V(s) ← max_a Σ P(s'|s,a) [R(s,a,s') + γ·V(s')]

Repeat until |ΔV| < θ (convergence threshold). Works by iterating the Bellman operator until fixed point.

🔁 Policy Iteration

Alternates between Policy Evaluation (compute V^π) and Policy Improvement (greedy update) until stable.

Eval: V^π(s) ← Σ P·[R + γ·V^π(s')] Improve: π(s) ← argmax_a Σ P·[R+γV]

Guaranteed to converge in finite steps for finite MDPs. Each improvement is monotonically better.
